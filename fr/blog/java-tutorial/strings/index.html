🔠 Qu'est-ce qu'une chaîne en Java ?

Une Chaîne est une séquence de caractères - comme "Hello" ou "123abc". C'est l'une des classes les plus utilisées en Java et elle est immuable, c'est-à-dire qu'une fois créée, elle ne peut pas être modifiée.

🛠️ Création de chaînes de caractères

String s1 = "Java";
String s2 = new String("Java");

Les deux fonctionnent, mais la première est préférable.

📦 Méthodes utiles pour les chaînes de caractères

. . . . .
Méthode Description
longueur() Retourne le nombre de caractères
charAt(index) Imprime le caractère à une position spécifique
toLowerCase() Convertit en minuscules
toUpperCase() Convertit en majuscules
trim() Supprime les espaces avant/arrière
substring(begin, end) Extrait d'une partie
contains(str) Vérifie si la sous-chaîne est présente
replace(a, b) Remplace les caractères ou les mots

🧪 Comparaison de chaînes de caractères

✅ Méthode correcte (compare les valeurs)

String a = "hello";
String b = "hello";
System.out.println(a.equals(b));  // true

❌ Wrong Way (compare les adresses mémoire)

System.out.println(a == b);  // sometimes true, sometimes false

Toujours utiliser .equals() pour comparer le contenu d'une chaîne.

🧬 Immutabilité des chaînes de caractères

String s = "Hi";
s.concat(" there");
System.out.println(s);  // Still "Hi"!

✅ Utilisez StringBuilder si vous avez besoin de modifier des chaînes de façon répétée.

🧱 StringBuilder et StringBuffer

StringBuilder (non sécurisé, plus rapide)

StringBuilder sb = new StringBuilder("Java");
sb.append(" Rocks!");
System.out.println(sb);  // Java Rocks!

StringBuffer (thread-safe)

StringBuffer sbf = new StringBuffer("Java");
sbf.append(" is cool");

🔍 Recherche dans les chaînes de caractères

String txt = "banana";
System.out.println(txt.indexOf("a"));      // 1
System.out.println(txt.lastIndexOf("a"));  // 5
System.out.println(txt.startsWith("ban")); // true
System.out.println(txt.endsWith("na"));    // true

🧹 Formatage des chaînes de caractères

String name = "Alice";
int age = 30;
System.out.printf("Name: %s, Age: %d\n", name, age);

Egalement disponible via String.format().

📘 Recap

  • Les chaînes de caractères sont des objets et sont immuables
  • Utiliser .equals() pour comparer des valeurs
  • Des tonnes de méthodes pratiques comme substring(), replace(), split()
  • Utiliser StringBuilder pour une meilleure performance dans les boucles
