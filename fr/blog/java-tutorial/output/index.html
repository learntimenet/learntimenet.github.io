📝 Ce que vous apprendrez

  • Utilisation de System.out.print() et println()
  • Formatage de la sortie avec printf()
  • Caractères d'échappement (comme \n, \t)
  • Impression des variables et des expressions

📤 Sortie de base

System.out.print("Hello ");
System.out.println("World!");

Sortie :

Hello World!
  • print() écrit du texte sans nouvelle ligne
  • println() ajoute une nouvelle ligne à la fin

✍️ Impression des variables

String name = "Java";
int year = 2025;
System.out.println("Welcome to " + name + " in " + year);

Sortie :

Welcome to Java in 2025

🧑‍💻 Sortie formatée avec printf()

La méthode printf() vous permet de contrôler la façon dont les valeurs sont affichées - similaire au formatage de style C.

System.out.printf("Name: %s, Age: %d, Score: %.2f", "Alice", 25, 95.678);

Sortie :

Name: Alice, Age: 25, Score: 95.68

Spécification de format commun

Tableau>

Format Description %s Chaîne %d Integer (décimal) %f Nombre à virgule flottante %.2f Deux décimales %n Nouvelle ligne (sûre pour la plateforme)

🔣 Caractères d'échappement

Tableau>

Code Sens \n Nouvelle ligne \t Tab \ Flash arrière \" Double quote 
System.out.println("Line1\nLine2");
System.out.println("Path: C:\\Java\\bin");

🧪 Expressions de sortie

System.out.println("2 + 3 = " + (2 + 3));

Sortie :

2 + 3 = 5

📘 Recap

  • Utilisez print() pour les retours à la ligne, println() pour les sauts de ligne.
  • printf() pour une sortie formatée et stylisée
  • Les caractères d'échappement ajoutent des tabulations, des nouvelles lignes, etc.
  • Combiner du texte + des variables avec + ou des spécificateurs de format
