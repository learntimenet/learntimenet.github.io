🔣 Que sont les opérateurs ?

Les opérateurs sont des symboles qui effectuent des opérations sur les variables et les valeurs, comme les mathématiques, les comparaisons et la logique.

Exemple :

int sum = 5 + 3; // '+' is an operator

➗ Opérateurs arithmétiques

Utilisé pour les mathématiques de base :

Opérateur Description Exemple
+ Addition a + b
- Soustraction a - b
* Multiplication a * b
/ Division a / b
% Module (reste) a % b 
int result = 10 % 3;  // result = 1

⚖️ Opérateurs de comparaison (relationnels)

Utilisé pour comparer deux valeurs :

Opérateur Sens Exemple
== Égal à a == b
!= Pas égal a != b
> Plus grand que a > ; b
< Moins de a < ; b
>= Grand ou égal a >= b
<= Inférieur ou égal a <= b

Elles renvoient vrai ou faux.

🔁 Opérateurs logiques

Utilisé avec les valeurs booléennes (vrai/faux) :

Opérateur Description Exemple
&& Logique ET a > ; 5 && ; b < ; 10
|| Or logique a > ; 5 || b < ; 10
! Logique NOT !(a > ; 5) 
boolean isAdult = age >= 18 && hasID;

🖊️ Opérateurs d'affectation

Attribuer des valeurs aux variables :

Opérateur Exemple Equivalent
= x = 5 x obtient 5
+= x += 3 x = x + 3
-= x -= 2 x = x - 2
*= x *= 4 x = x * 4
/= x /= 2 x = x / 2

🧠 Prédominance de l'opérateur

Tout comme les mathématiques, Java suit un ordre d'opérations :

  1. ()
  2. *, /, %
  3. +, -
  4. Comparaisons (>, <, etc.)
  5. Logique (&&, ||)
  6. Affectation (=)

Exemple :

int result = 2 + 3 * 4; // result = 14, not 20!

Utilisez () pour contrôler l'ordre :

int result = (2 + 3) * 4; // result = 20

🔎 Exemple : Extrait de calculatrice

public class Calculator {
    public static void main(String[] args) {
        int a = 10, b = 4;

        System.out.println("Sum: " + (a + b));
        System.out.println("Product: " + (a * b));
        System.out.println("Is a > b? " + (a > b));
    }
}

📘 Recap

  • Les opérateurs sont essentiels pour construire des expressions et des logiques de décision.
  • Java prend en charge les opérateurs arithmétiques, de comparaison, logiques et d'affectation.
  • Utilisez les parenthèses () pour clarifier les expressions complexes.
