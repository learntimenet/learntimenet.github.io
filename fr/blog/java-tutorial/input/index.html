📝 Ce que vous apprendrez

  • Utilisation de Scanner pour lire l'entrée
  • Lecture de chaînes de caractères, d'entiers, de doubles et de booléens
  • Traitement des questions liées à la saisie des données

🧰 Importation du scanner

Pour utiliser les fonctionnalités d'entrée, vous devez importer la classe Scanner de java.util.

import java.util.Scanner;

🔡 Exemple de base

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter your name: ");
        String name = sc.nextLine();
        
        System.out.println("Hello, " + name + "!");
    }
}

✅ nextLine() lit une ligne complète d'entrée.

🔢 Lire les nombres

System.out.print("Enter your age: ");
int age = sc.nextInt();  // Reads an integer

System.out.print("Enter your height: ");
double height = sc.nextDouble();  // Reads a decimal

📌 Autres types d'entrées

Méthode Type d'entrée
next() Mot unique
nextLine() Ligne entière
nextInt() Integer
nextDouble() Décimale
nextBoolean() true / false

⚠️ Piège courant : Mélanger nextInt() et nextLine()

int age = sc.nextInt();
sc.nextLine(); // Consume leftover newline
String name = sc.nextLine();

Toujours effacer le tampon après nextInt() ou nextDouble() avant d'utiliser nextLine().

🧪 Exemple de programme

System.out.print("Enter a number: ");
int num = sc.nextInt();
System.out.println("Square: " + (num * num));

🧼 Fermeture du scanner

N'oubliez pas de fermer le scanner lorsque vous avez terminé :

sc.close();

📘 Recap

  • Utiliser Scanner pour l'entrée de la console
  • Choisir la bonne méthode pour le type de données
  • Utilisez nextLine() pour les phrases complètes
  • Attention aux problèmes de tampon d'entrée !
