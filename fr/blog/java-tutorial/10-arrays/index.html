🧱 Qu'est-ce qu'un tableau ?

Un tableau est un conteneur qui contient un nombre fixe de valeurs du même type.

Il s'agit d'une rangée de boîtes :

int[] numbers = {10, 20, 30, 40};

🔧 Déclarer des tableaux

Il existe plusieurs façons de déclarer des tableaux en Java :

int[] scores;      // preferred
String names[];    // also valid

Puis initialiser :

scores = new int[5];  // space for 5 integers

Ou déclarer et initialiser en une seule ligne :

int[] ages = {18, 21, 25, 30};

📥 Accès aux éléments du tableau

Les tableaux sont zéro-indexés :

System.out.println(ages[0]);  // 18
ages[2] = 26;                 // update value

🔁 Boucler des tableaux

Utilisez une boucle for pour parcourir chaque élément :

for (int i = 0; i < ages.length; i++) {
    System.out.println(ages[i]);
}

Ou une boucle foreach :

for (int age : ages) {
    System.out.println(age);
}

📐 Longueur du tableau

Utilisez .length pour obtenir le nombre d'éléments :

System.out.println("Total: " + ages.length);

🔢 Tableaux multidimensionnels

Vous pouvez créer des tableaux en 2D (ou plus) :

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};

System.out.println(matrix[1][2]);  // Output: 6

Boucle dans les tableaux 2D :

for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}

🧠 Exemple : Calculatrice moyenne

public class Average {
    public static void main(String[] args) {
        int[] marks = {85, 90, 78, 92};
        int sum = 0;

        for (int mark : marks) {
            sum += mark;
        }

        double average = sum / (double) marks.length;
        System.out.println("Average: " + average);
    }
}

⚠️ Pièges courants

  • Accès aux indices hors limites : array[array.length] → ❌
  • Les tableaux ne peuvent pas croître/rétrécir - utilisez ArrayList pour les collections dynamiques (plus d'informations à ce sujet ultérieurement).

📘 Recap

  • Les tableaux stockent plusieurs valeurs du même type
  • L'accès aux éléments se fait par l'index (en commençant par 0)
  • Utiliser des boucles pour traiter facilement des tableaux
  • Java prend en charge les tableaux multidimensionnels
