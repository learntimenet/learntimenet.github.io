🧠 Qu'est-ce que le cadre des collections ?

Le Collections Framework est un ensemble de classes et d'interfaces dans java.util qui vous aide à travailler avec des groupes d'objets comme des listes de noms, des ensembles de tags, ou des cartes d'utilisateurs et de mots de passe.

🔑 Core Interfaces

.
Interface Description Exemple de classe
Liste Collection ordonnée, autorise les doublonsArrayList, LinkedList
Set Non ordonné, pas de doublons HashSet, TreeSet
Carte Paires clé-valeur HashMap, TreeMap
Queue Collecte premier entré, premier sorti LinkedList, PriorityQueue

📋 Travailler avec des listes

List<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");
System.out.println(fruits.get(0));  // Apple

Utilisez ArrayList pour un accès rapide, LinkedList pour des insertions/suppressions rapides.

🧩 Ensembles - Pas de doublons

Set<String> colors = new HashSet<>();
colors.add("Red");
colors.add("Green");
colors.add("Red");  // ignored
System.out.println(colors);  // [Red, Green]

Utilisez TreeSet si vous souhaitez un tri automatique.

🔑 Cartes - Paires clé-valeur

Map<String, Integer> scores = new HashMap<>();
scores.put("Alice", 90);
scores.put("Bob", 85);
System.out.println(scores.get("Alice"));  // 90

✅ HashMap pour une utilisation générale ✅ LinkedHashMap pour préserver l'ordre d'insertion ✅ TreeMap pour les clés triées

📦 Files d'attente - FIFO

Queue<String> tasks = new LinkedList<>();
tasks.offer("Task1");
tasks.offer("Task2");
System.out.println(tasks.poll());  // Task1

Utilisez PriorityQueue pour trier automatiquement les éléments par priorité.

🔁 Itération sur les collections

for (String fruit : fruits) {
    System.out.println(fruit);
}

Ou en utilisant un itérateur :

Iterator<String> it = fruits.iterator();
while (it.hasNext()) {
    System.out.println(it.next());
}

🧠 Choisir la bonne collection

Nécessité Utiliser ceci
Accès aléatoire rapide ArrayList
Pas de doublons HashSet
Éléments triés TreeSet, TreeMap
Accès rapide par touches HashMap
Traitement FIFO Queue

📘 Recap

  • Le cadre des collections comprend des outils puissants pour la gestion des données
  • Connaître les interfaces de base : List, Set, Map, et Queueue.
  • Chacun d'entre eux dispose de plusieurs implémentations optimisées pour différents cas d'utilisation.
  • Apprendre à choisir la bonne pour la performance et la clarté
