🧱 Structure de base de Java

Chaque application Java est composée de classes et de méthodes. La méthode main est le point d'entrée de tout programme Java autonome.

Voici un programme Java minimal :

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

🔍 Ventilation

  • public classe HelloWorld : Définit une classe nommée HelloWorld. Les programmes Java sont basés sur des classes.
  • public static void main(String[] args) : Le point de départ du programme.
  • System.out.println() : Une instruction qui imprime sur la console.
  • {} : Blocs de code regroupés.
  • ; : Chaque instruction se termine par un point-virgule.

🧭 Java est sensible à la casse

Ce ne sont pas les mêmes :

System.out.println();    // valid
system.out.println();    // invalid – “system” should be capitalized

🧠 Conventions d'appellation

Suivez ces conventions standard pour que le code reste lisible :

| Type | Convention | Exemple | |-||--| | Classe | PascalCase | MonPremierClasse | | méthode/variable| camelCase | printMessage() | | Constant | ALL_CAPS_WITH_UNDERSCORES | MAX_VALUE |

✅ Conseils pour le formatage du code

  • Utilisez l'indentation (généralement 4 espaces ou une tabulation).
  • Gardez les accolades sur la même ligne ou sur la ligne suivante (en fonction des paramètres de votre IDE).
  • Ajouter des commentaires en utilisant :
    • // pour une seule ligne
    • /* */ pour les lignes multiples
// This is a single-line comment

/*
 This is a
 multi-line comment
*/

💡 Exemple : Programme Java simple

public class Greeting {
    public static void main(String[] args) {
        String name = "Java";
        System.out.println("Hello, " + name + "!");
    }
}

🚫 Erreurs de syntaxe communes

  • Point-virgule manquant ;
  • Accolades {} ou parenthèses () non appariées
  • Mauvaise casse (par exemple, String vs string)
  • Noms de méthodes ou de classes incorrects

📘 Recap

Le code Java suit une structure cohérente :

  • Les classes contiennent des méthodes
  • La méthode main() est le point de départ.
  • Chaque déclaration se termine par ;
  • Les accolades définissent les blocs de code
