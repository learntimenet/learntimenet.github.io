🔄 Qu'est-ce qu'une boucle While ?

Une bouclewhile continue de fonctionner tant que sa condition est vraie.

🔧 Syntaxe :

while (condition) {
    // code to repeat
}

✅ Exemple

int i = 1;

while (i <= 5) {
    System.out.println("i = " + i);
    i++;
}

🔁 Ceci imprime i = 1 à i = 5.

🔂 La boucle do... while (faire... pendant)

Cette version s'exécute toujours au moins une fois - parce que la condition est vérifiée après la première exécution.

🔧 Syntaxe :

do {
    // code to run
} while (condition);

Exemple :

int i = 1;

do {
    System.out.println("i = " + i);
    i++;
} while (i <= 5);

🔍 Quand utiliser Which ?

Utiliser ceci... Quand...
while Vous pouvez sauter la boucle entièrement
do...while Vous voulez toujours qu'il s'exécute au moins une fois

⚠️ Erreurs courantes

  • Oublier de mettre à jour la variable de la boucle (boucle infinie !)
  • Mauvais placement des points-virgules : ``Java while (i < 5) ; // ❌ Ne pas faire ça
  • Utiliser l'assignation au lieu de la comparaison : ``Java while (x = 5) // ❌ devrait être while (x == 5)

🛠 Exemple concret

Simulation de réessai de mot de passe:

Scanner scanner = new Scanner(System.in);
String password;

do {
    System.out.print("Enter password: ");
    password = scanner.nextLine();
} while (!password.equals("java123"));

System.out.println("Access granted!");

📘 Recap

  • Les boucles while vérifient la condition avant de s'exécuter
  • Les boucles do...while s'exécutent au moins une fois
  • Attention aux boucles infinies et à la logique des conditions
