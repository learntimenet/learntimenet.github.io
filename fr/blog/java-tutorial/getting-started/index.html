🔧 Qu'est-ce que le JDK ?

Le Java Development Kit (JDK) est un logiciel qui comprend le compilateur Java, l'environnement d'exécution Java (JRE) et les outils permettant de développer des programmes Java.

Vous devez l'installer pour :

  • Compiler les fichiers .java en bytecode
  • Exécuter des applications Java localement

⬇️ Étape 1 : Télécharger le JDK

  1. Allez sur le site officiel de Java : 👉 https://dev.java

  2. Choisissez votre système d'exploitation (Windows, macOS, Linux)

  3. Téléchargez la dernière version Long-Term Support (LTS) - actuellement Java 21 LTS est recommandé.

💻 Étape 2 : Installer le JDK

Sous Windows :

  • Exécuter le programme d'installation .exe.
  • Accepter les valeurs par défaut ou choisir le chemin d'installation (par exemple, C:\NProgram Files\NJava\Njdk-21)
  • Après l'installation, définissez votre variable d'environnement JAVA_HOME.
  • Ajouter JAVA_HOME\bin au PATH du système

Sur macOS :

  • Installer en utilisant le programme d'installation .pkg.
  • Java est généralement configuré automatiquement par macOS.
  • Vérifier via le terminal : java -version

Sur Linux (Ubuntu/Debian) :

sudo apt update
sudo apt install openjdk-21-jdk

✅ Étape 3 : Vérifier l'installation

Ouvrez votre terminal ou votre invite de commande et tapez

java -version

Vous devriez voir quelque chose comme :

openjdk version "21.0.2" 2024-01-16
OpenJDK Runtime Environment (build 21.0.2+13-58)
OpenJDK 64-Bit Server VM (build 21.0.2+13-58, mixed mode, sharing)

🛠️ Étape 4 : Choisir un IDE

Un IDE (Integrated Development Environment) facilite l'écriture de Java grâce à des fonctionnalités telles que l'autocomplétion et le débogage.

IDEs populaires :

  • IntelliJ IDEA (Community Edition) - La meilleure option complète
  • VS Code - Léger avec extensions Java
  • Eclipse - Option vétéran, un peu désuète mais fonctionnelle

👉 Pour les débutants, nous recommandons IntelliJ :

  • [Télécharger IntelliJ IDEA] (https://www.jetbrains.com/idea/download/)

✏️ Étape 5 : Créer votre premier fichier Java

Dans votre IDE ou votre éditeur de texte :

  1. Créez un nouveau fichier appelé HelloWorld.java
  2. Ajoutez ce code :
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  1. Compilez le fichier (si vous utilisez le terminal) :
javac HelloWorld.java
  1. Exécutez-le :
java HelloWorld

🎉 Sortie

Hello, World!

Si vous voyez cela, vous êtes prêt à vous lancer dans la programmation Java !
