🧭 Déclarations conditionnelles

Ils permettent à votre programme de prendre des décisions.

✅ if, else if, else

int score = 85;

if (score >= 90) {
    System.out.println("Grade: A");
} else if (score >= 80) {
    System.out.println("Grade: B");
} else {
    System.out.println("Grade: C or lower");
}

🔄 switch Statement

La meilleure solution consiste à vérifier une variable par rapport à de nombreuses valeurs possibles.

int day = 3;

switch (day) {
    case 1: System.out.println("Monday"); break;
    case 2: System.out.println("Tuesday"); break;
    case 3: System.out.println("Wednesday"); break;
    default: System.out.println("Other day");
}

⚠️ Utiliser break pour éviter le fall-through (exécution de plusieurs cas).

🔁 Boucles

Les boucles vous permettent de répéter le code tant qu'une condition est vraie.

➰ for Loop (Count-Controlled)

for (int i = 1; i <= 5; i++) {
    System.out.println("Count: " + i);
}

🔄 Boucle while (contrôlée par des conditions)

int i = 1;
while (i <= 5) {
    System.out.println("Count: " + i);
    i++;
}

🔂 Boucle do-while (s'exécute au moins une fois)

int i = 1;
do {
    System.out.println("Count: " + i);
    i++;
} while (i <= 5);

🧨 break et continue

break : sortir prématurément d'une boucle

for (int i = 1; i <= 10; i++) {
    if (i == 5) break;
    System.out.println(i);
}

continue : sauter l'itération de la boucle en cours

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);  // 3 is skipped
}

🧠 Exemple : Logique de tentative de connexion

String password = "java123";
int attempts = 0;
boolean loggedIn = false;

while (attempts < 3 && !loggedIn) {
    // simulate input
    String input = "java123";
    if (input.equals(password)) {
        System.out.println("Welcome!");
        loggedIn = true;
    } else {
        System.out.println("Wrong password.");
        attempts++;
    }
}

📘 Recap

  • Utiliser if, else, et switch pour prendre des décisions
  • Utiliser des boucles pour répéter le code : for, while, do-while.
  • Contrôler le comportement de la boucle avec break et continue.
