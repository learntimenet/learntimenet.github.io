🔧 Que sont les opérateurs ?

Les opérateurs sont des symboles spéciaux ou des mots-clés utilisés pour effectuer des opérations sur des valeurs ou des variables. Ils constituent l'épine dorsale de la logique et des mathématiques dans les programmes JavaScript.

Exemple :

let total = 5 + 3;

Ici, + est un opérateur qui additionne deux nombres.

➕ Opérateurs arithmétiques

Utilisé pour les mathématiques de base :

| Opérateur | Description | Exemple | |-||--| | + | Addition | 5 + 2 → 7 | | - | Soustraction | 5 - 2 → 3 | | * | Multiplication | 5 * 2 → 10 | | / | Division | 10 / 2 → 5 | | % | Modulus (reste) | 5 % 2 → 1 | | ** | Exponentiation | 2 ** 3 → 8 |

🧾 Opérateurs d'affectation

Permet d'attribuer des valeurs aux variables :

Opérateur Exemple Sens
= x = 10 Assigner 10 à x
+= x += 5 x = x + 5
-= x -= 3 x = x - 3
*= x *= 2 x = x * 2
/= x /= 2 x = x / 2
%= x %= 2 x = x % 2

🔍 Opérateurs de comparaison

Utilisé pour comparer des valeurs (résultat vrai ou faux) :

Opérateur Description Exemple
== Égal à 5 == "5" → true
=== Valeur égale & ; type 5 === "5" → false
!= Pas égal 5 != 3 → true
!== Pas égal (valeur/type) 5 !== "5" → true
> Plus grand que 5 > ; 3 → true
< Moins de 5 < ; 3 → false
>= Plus grand ou égal 5 >= 5 → true
<= Inférieur ou égal 4 <= 5 → true

⚙️ Opérateurs logiques

Utilisé pour combiner ou inverser des valeurs booléennes :

Opérateur Description Exemple
&& Logique ET vrai && ; faux → faux
` ` Or logique `true false→true`
! Logique NOT !true → false
