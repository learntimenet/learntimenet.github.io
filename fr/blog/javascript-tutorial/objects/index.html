🔍 Qu'est-ce qu'un objet ?

Un objet est une collection de paires clé-valeur où chaque clé est un identifiant unique (également appelé propriété ou attribut) et chaque valeur peut être n'importe quel type de données, y compris d'autres objets. Les objets sont l'une des structures de données les plus importantes en JavaScript et sont largement utilisés pour représenter des entités du monde réel.

Syntaxe de base

let person = {
  name: "Alice",
  age: 30,
  isActive: true
};

Ici, personne est un objet avec trois propriétés :

  • name : une chaîne de caractères
  • age : un nombre
  • isActive : un booléen

🛠️ Accès aux propriétés des objets

Vous pouvez accéder aux propriétés d'un objet en utilisant soit la notation dot, soit la notation bracket.

Notation par points :

console.log(person.name);  // Output: Alice
console.log(person.age);   // Output: 30

Notation de la parenthèse :

La notation entre crochets est utile lorsque les noms de propriétés sont dynamiques ou ne sont pas des identifiants JavaScript valides (par exemple, lorsqu'ils contiennent des espaces).

console.log(person["name"]);  // Output: Alice
console.log(person["age"]);   // Output: 30

🔄 Modifier les propriétés d'un objet

Vous pouvez modifier la valeur d'une propriété d'un objet en lui attribuant simplement une nouvelle valeur.

Exemple :

person.age = 31;  // Modify the age
person["isActive"] = false;  // Modify the isActive property

console.log(person.age);  // Output: 31
console.log(person.isActive);  // Output: false

🧑‍💻 Ajouter de nouvelles propriétés aux objets

Vous pouvez également ajouter de nouvelles propriétés à un objet après sa définition.

person.city = "New York";  // Add a new property

console.log(person.city);  // Output: New York

🔨 Méthodes dans les objets

Les objets peuvent également avoir des méthodes, qui sont des fonctions stockées en tant que propriétés de l'objet.

Exemple :

let person = {
  name: "Alice",
  greet: function() {
    console.log("Hello, " + this.name);
  }
};

person.greet();  // Output: Hello, Alice

Ici, la propriété greet est une méthode qui imprime un message de bienvenue en utilisant la propriété name. Nous avons utilisé this pour faire référence à l'objet courant.

🏷️ Destructuration d'objets

**La restructuration vous permet d'extraire plusieurs propriétés d'un objet et de les affecter à des variables de manière plus concise.

Exemple :

let person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

let { name, age, city } = person;

console.log(name);  // Output: Alice
console.log(age);   // Output: 30
console.log(city);  // Output: New York

🛠️ Objets imbriqués

Les objets peuvent contenir d'autres objets, ce qui permet de créer des structures de données plus complexes.

Exemple :

let person = {
  name: "Alice",
  address: {
    street: "123 Main St",
    city: "New York"
  }
};

console.log(person.address.city);  // Output: New York

Ici, address est un objet à l'intérieur de l'objet person. Vous pouvez accéder aux propriétés imbriquées en utilisant la notation par points.

🧠 Recap

  • Les objets stockent des paires clé-valeur et sont utilisés pour représenter des entités du monde réel.
  • Vous pouvez accéder, modifier et ajouter des propriétés en utilisant la notation par points ou par crochets.
  • Les méthodes** des objets sont des fonctions qui font partie de l'objet.
  • La déstructuration des objets permet d'accéder aux propriétés de manière plus concise.
  • Les objets peuvent être enchevêtrés, c'est-à-dire qu'ils peuvent contenir d'autres objets.
