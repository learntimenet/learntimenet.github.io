JavaScript est un langage de programmation interprété et léger utilisé le plus souvent pour créer des sites web interactifs. Il fonctionne avec HTML et CSS pour créer des expériences utilisateur riches.

Il est supporté par tous les navigateurs web modernes, et peut être utilisé à la fois pour le développement frontend et backend.

⚙️ Comment fonctionne JavaScript

Le code JavaScript est exécuté par le moteur JavaScript du navigateur. Il peut :

  • Répondre aux actions de l'utilisateur (comme les clics et les saisies au clavier)
  • Modifier HTML et CSS de manière dynamique
  • Envoyer et recevoir des données à l'aide d'API

🚀 Pourquoi apprendre JavaScript ?

  • C'est le langage de programmation le plus utilisé pour le web.
  • Permet une interaction en temps réel dans les applications web
  • Fonctionne avec des frameworks populaires tels que React, Angular et Vue.
  • Peut également être utilisé côté serveur avec Node.js

🛠 Écrire son premier scénario

Vous pouvez écrire du JavaScript directement dans un fichier HTML à l'aide de la balise <script> :

<!DOCTYPE html>
<html>
  <head>
    <title>My First JS</title>
  </head>
  <body>
    <h1>Hello JavaScript!</h1>
    
    <script>
      alert("Welcome to JavaScript!");
    </script>
  </body>
</html>

Lorsque vous ouvrez ce fichier HTML dans un navigateur, un message contextuel apparaît.

✍️ Aperçu de la syntaxe de base

La syntaxe JavaScript est simple et flexible. Voici un exemple de déclaration de variable et de fonction simple :

let name = "Alex";

function greet(user) {
  console.log("Hello, " + user + "!");
}

greet(name);

Sortie dans la console:

Hello, Alex!

📦 Principales caractéristiques de JavaScript

  • Type dynamique - Pas besoin de déclarer les types de données
  • Fonctions de première classe - Les fonctions sont traitées comme des variables
  • Event-Driven - Conçu pour les interactions avec les utilisateurs
  • Programmation asynchrone** - Prise en charge des promesses, async/await

🔍 Où JavaScript est utilisé

  • Développement web (frontend et backend)
  • Développement d'applications mobiles (par exemple, React Native)
  • Applications de bureau (par exemple, Electron)
  • Développement de jeux
  • Informatique sans serveur

⚠️ Les pièges les plus courants pour les débutants

  • Oublier d'utiliser let, const, ou var pour les déclarations de variables
  • Incompréhension de == vs. === (égalité stricte)
  • Surutilisation de alert() et console.log() pour le débogage
  • Ignorer la compatibilité des navigateurs
