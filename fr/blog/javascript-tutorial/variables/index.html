🔤 Qu'est-ce qu'une variable ?

Une variable est un conteneur qui stocke des données. Vous pouvez l'assimiler à une boîte qui contient une valeur - comme un nom, un nombre ou même une liste.

let name = "Alex";

Dans l'exemple ci-dessus, la variable name contient la valeur "Alex".

✍️ Déclarer des variables

Vous pouvez déclarer des variables en utilisant l'un des trois mots-clés suivants : var, let, ou const.

✅ let

Introduit dans ES6, let est la façon la plus courante de déclarer des variables qui peuvent changer plus tard.

let age = 25;
age = 26;

🔒 const

Utilisez const lorsque vous voulez une valeur constante qui ne doit pas changer.

const pi = 3.14;
// pi = 3.15; ❌ This will cause an error

📦 var (ancienne méthode)

var est l'ancienne façon de déclarer des variables et est généralement évitée dans le JavaScript moderne en raison de problèmes de portée.

var greeting = "Hello!";

📛 Règles de désignation des variables

Lors de la désignation d'une variable :

  • Les noms doivent commencer par une lettre, _ ou $.
  • Peut contenir des lettres, des chiffres, _ et $.
  • Ne peut pas utiliser de mots réservés (par exemple, let, class, function)
  • Sensible à la casse (name et Name sont différents)

✅ Valide : userName, totalAmount, $price, _temp ❌ Invalid : 1stPlace, var, nom d'utilisateur

🧠 Attribution ultérieure de valeurs

Vous pouvez déclarer une variable sans lui attribuer immédiatement une valeur :

let score;
score = 100;

📘 Recap

  • Utilisez let pour les variables qui changent
  • Utiliser const pour les valeurs fixes
  • Évitez var à moins que vous n'ayez affaire à du code plus ancien
  • Nommer les variables de manière claire et cohérente
