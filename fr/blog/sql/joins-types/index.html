Introduction

Dans cet article, nous allons apprendre les types de jointures SQL, y compris inner join, left join, right join et full outer join. Nous couvrirons leur syntaxe, leurs différences et des exemples pratiques pour une meilleure compréhension.

INNER JOIN

Combine les lignes de deux tables en fonction d'une colonne commune entre elles

A
B 
SELECT *
FROM A
INNER JOIN B ON A.key = B.key

LEFT JOIN

Renvoie toutes les lignes de la table de gauche et les lignes correspondantes de la table de droite en fonction d'une condition spécifiée. S'il n'y a pas de correspondance dans la table de droite, des valeurs NULL sont retournées pour les colonnes de la table de droite

A
B 
SELECT *
FROM A
LEFT JOIN B ON A.key = B.key

LEFT EXCLUDING JOIN

Renvoie toutes les lignes de la table de gauche qui ne correspondent à aucune ligne de la table de droite

A
B 
SELECT *
FROM A
LEFT JOIN B ON A.key = B.key
WHERE B.key IS NULL

RIGHT JOIN

Renvoie toutes les lignes de la table de droite et les lignes correspondantes de la table de gauche en fonction d'une condition spécifiée. S'il n'y a pas de correspondance dans la table de gauche, des valeurs NULL sont retournées pour les colonnes de la table de gauche

A
B 
SELECT *
FROM A
RIGHT JOIN B ON A.key = B.key

RIGHT EXCLUDING JOIN

Renvoie toutes les lignes de la table de droite qui ne correspondent à aucune ligne de la table de gauche.

B 
SELECT *
FROM A
RIGHT JOIN B ON A.key = B.key
WHERE A.key IS NULL

OUTER JOIN

Combine les lignes de deux ou plusieurs tables en fonction d'une condition spécifiée, et inclut les lignes non correspondantes d'une ou des deux tables
