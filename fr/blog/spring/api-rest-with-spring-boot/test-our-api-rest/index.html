Conclusion

Nous avons maintenant testé notre API REST en utilisant Postman, en vérifiant que nous pouvons créer, récupérer, mettre à jour et supprimer des tâches Todo avec succès.

Cependant, nous avons constaté que la création de tâches Todo sans titre ni description est possible. Dans la prochaine section, nous allons ajouter une couche de validation pour corriger ce comportement et garantir que les données respectent les contraintes définies.
