Introduction

Angular est un framework front-end populaire pour la création d'applications web dynamiques. Cet aide-mémoire fournit une référence rapide pour les commandes, la syntaxe et les meilleures pratiques d'Angular.

Commandes de l'interface de programmation d'Angular

Le CLI Angular est un outil puissant pour l'échafaudage et la gestion des projets Angular.

cheatsheet

  • Crée un nouveau projet Angular
    . ng nouveau nom de projet
  • Démarre un serveur de développement (http://localhost:4200)
    ng servir
  • Exécute l'application sur un port personnalisé
    . ng serve --port 5000
  • Compile l'application dans le dossier dist/
    . ng construire
  • Version de production (optimisée)
    ng build --prod
  • Exécute des tests unitaires (Karma + Jasmine)
    ng test
  • Exécute des tests de bout en bout (Protractor)
    ng e2e
  • Génère un nouveau composant
    ng generate component my-component
  • Génère un nouveau service
    ng generate service my-service
  • Génère un nouveau module
    ng generate module my-module
  • Génère une nouvelle directive
    ng generate directive my-directive
  • Génère un nouveau tuyau
    ng generate pipe my-pipe
  • Ajoute Angular Material au projet
    . ng add @angular/material
  • Mise à jour des paquets Angular
    mise à jour des ng
  • Exécute le linting (TSLint/ESLint)
    peluches ng

Composants de base

Syntaxe du modèle

<!-- Interpolation -->
<p>{{ title }}</p>

<!-- Property Binding -->
<img [src]="imageUrl">

<!-- Event Binding -->
<button (click)="onClick()">Click Me</button>

<!-- Two-Way Binding (requires FormsModule) -->
<input [(ngModel)]="name">

Structure des composants

import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.html',
  styleUrls: ['./my-component.css']
})
export class MyComponent {
  title = 'My App';
  
  onClick() {
    console.log('Button clicked!');
  }
}

Directives

Directives intégrées

cheatsheet

  • Rendu conditionnel d'un élément
    . *ngIf
  • Boucle sur une liste
    *ngFor
  • Application dynamique des classes CSS
    [ngClass]
  • Application dynamique des styles en ligne
    . [ngStyle]
  • Cas de rechange pour les modèles
    [ngSwitch]

Exemple :

<div *ngIf="isLoggedIn">Welcome back!</div>

<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>

<div [ngClass]="{ 'active': isActive }"></div>

<div [ngStyle]="{ 'color': textColor }"></div>

<div [ngSwitch]="role">
  <p *ngSwitchCase="'admin'">Admin Panel</p>
  <p *ngSwitchCase="'user'">User Dashboard</p>
  <p *ngSwitchDefault>Guest</p>
</div>

Services et injection de dépendance

Création d'un service

import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root' // Singleton service
})
export class DataService {
  getData() {
    return ['Item 1', 'Item 2', 'Item 3'];
  }
}

Utilisation d'un service dans un composant

import { Component } from '@angular/core';
import { DataService } from './data.service';

@Component({...})
export class MyComponent {
  items: string[];

  constructor(private dataService: DataService) {
    this.items = this.dataService.getData();
  }
}

Client HTTP (appels API)

Importer HttpClientModule dans app.module.ts :

import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule],
})
export class AppModule {}

Utiliser HttpClient dans un Service :

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable()
export class ApiService {
  constructor(private http: HttpClient) {}

  getUsers() {
    return this.http.get('https://api.example.com/users');
  }

  postData(data: any) {
    return this.http.post('https://api.example.com/data', data);
  }
}

S'abonner dans un composant :

this.apiService.getUsers().subscribe(users => {
  console.log(users);
});

Routage et navigation

Définir les routes dans app-routing.module.ts :

import { RouterModule, Routes } from '@angular/router';
import { HomeComponent } from './home.component';
import { AboutComponent } from './about.component';

const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: '**', redirectTo: '' } // Wildcard route (404)
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule {}

Ajouter <router-outlet> dans app.component.html :

<router-outlet></router-outlet>

Naviguer par programme :

import { Router } from '@angular/router';

constructor(private router: Router) {}

goToAbout() {
  this.router.navigate(['/about']);
}

Lien dans le modèle :

<a routerLink="/">Home</a>
<a routerLink="/about">About</a>

Formulaires

Formulaires basés sur des modèles

Importez FormsModule dans votre app.module.ts :

import { FormsModule } from '@angular/forms';

@NgModule({
  imports: [FormsModule],
})
export class AppModule {}

Utilisez ngModel et ngSubmit dans vos composants : 

<form #myForm="ngForm" (ngSubmit)="onSubmit(myForm)">
  <input name="username" ngModel required>
  <button type="submit">Submit</button>
</form>

Formes réactives

Importez ReactiveFormsModule dans votre app.module.ts :

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [ReactiveFormsModule],
})
export class AppModule {}

Créez votre FormGroup dans votre composant :

export class MyFormComponent {
  form = new FormGroup({
    username: new FormControl('', Validators.required),
    email: new FormControl('', [Validators.required, Validators.email])
  });

  onSubmit() {
    console.log(this.form.value);
  }
}

Et utilisez-le dans le code html de votre composant :

<form [formGroup]="form" (ngSubmit)="onSubmit()">
  <input formControlName="username">
  <input formControlName="email">
  <button type="submit">Submit</button>
</form>
