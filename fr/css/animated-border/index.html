Introduction

  • style.css

    .border-animation {
  position: relative;
  padding: 20px 30px;
  overflow: hidden;

.border-top {
  position: absolute;
  top: 0;
  right: 0;
  width: 100%;
  height: 3px;
  background: linear-gradient(to right, #c72b79, #de6d08, #fcc23f);
  animation: animate-border-top 2s linear infinite;
}

@keyframes animate-border-top {
  0% {
    transform: translateX(-100%);
  }

  100% {
    transform: translateX(100%);
  }
}

  animation-delay: 1s;
