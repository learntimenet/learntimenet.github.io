Description

Outil en ligne gratuit pour sélectionner facilement une couleur et construire une palette de couleurs.

Le design et les animations de ce site web sont principalement inspirés du site coolors, et nous tenons à remercier ses créateurs pour leur excellent travail !

Une fois une couleur sélectionnée, les formats de couleur (HEX, RGB, HSV, ...) et les variations de couleur (Nuances, Teintes, Tons, ...) seront affichés. Au survol, deux boutons apparaîtront :

  •    qui vous permet de copier le code HEX de la couleur dans votre presse-papiers
  •    qui vous permet d'ajouter cette couleur à votre palette

Pour réorganiser votre palette, vous pouvez utiliser la fonction Glisser-Déposer pour changer la position des couleurs. Personnellement, nous trouvons cette fonctionnalité très utile, et nous espérons que vous l'apprécierez également.

Pour de nouvelles inspirations, vous pouvez générer de nouvelles couleurs en cliquant sur le bouton situé dans le coin supérieur droit de la couleur sélectionnée et dans la section de la palette.

Nous espérons que vous apprécierez cet outil 😊
