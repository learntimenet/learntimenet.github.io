Description

Un outil en ligne qui vous aidera à apprendre la dactylographie (aussi appelée frappe à l'aveugle), une méthode qui vous permet de taper sans utiliser la vue pour trouver les touches du clavier. Chaque fois que vous regardez le clavier ou faites une erreur, vous ralentissez votre vitesse de frappe. Pour être plus productif, il serait idéal de taper rapidement et avec précision, et c'est ce que nous vous aiderons à faire sur ce site.

De nombreuses applications ont été développées pour aider les utilisateurs à améliorer leurs compétences en dactylographie. Notre différence avec les autres applications est que vous pouvez améliorer vos compétences en tapant de magnifiques citations et poèmes, et si vous êtes un ingénieur logiciel comme moi, vous pouvez taper du code source en Java ou en Python. Une autre fonctionnalité importante du site est la possibilité de voir votre vitesse de frappe et votre précision, ainsi que votre progression. Chaque score de test de frappe sera enregistré, et vous pourrez suivre vos progrès.

La convention pour utiliser un clavier est de placer vos doigts sur la rangée de repos, celle qui contient les touches F et J. La majorité des gens tapent entre 35 et 40 mots par minute (WPM), ce qui équivaut à 190 et 200 caractères par minute (CPM). Cependant, les dactylographes professionnels tapent beaucoup plus vite, dépassant les 75 WPM. La vitesse de frappe s'améliore généralement avec la pratique. Pendant l'entraînement, il est important de s'assurer qu'il n'y a pas de touches faibles. La vitesse de frappe est généralement déterminée par la lenteur de ces touches faibles plutôt que par la vitesse à laquelle les autres touches sont tapées. C'est pourquoi il est conseillé de commencer par nos leçons d'entraînement pour mémoriser la position des touches. Par défaut, le clavier Qwerty est sélectionné, et si vous utilisez un clavier Azerty, vous pouvez modifier vos préférences pour obtenir les leçons adaptées.
