1. Download the Java Development Kit (JDK)

To start coding in Java, you'll need the Java Development Kit (JDK), which includes the tools necessary to compile and run Java program. You can download the latest version of the JDK from the official source: https://jdk.java.net/. Choose the appropriate version for your operating system (Windows, macOS, or Linux).

2. Install the JDK

After downloading, follow the installation instructions specific to your O. For Windows users, this typically involves running the installer and following the prompt. On macOS and Linux, you might need to use terminal commands to install and configure the JDK.

3. Set Up Environment Variables

Setting up environment variables allows your system to recognize Java commands from any terminal window.

  • JAVA_HOME: This variable should point to the directory where the JDK is installed.

  • Path: Add the bin directory inside your JDK installation to the system's PATH variable.

This setup ensures that you can compile and run Java programs from the command line.

4. Verify the Installation

To confirm that Java is correctly installed, open your terminal or command prompt and type:

java --versin

You should see output indicating the installed Java version.

‚úçÔ∏è Writing Your First Java Program

Let's create a simple Java program that prints Hello, World! to the console.

1. Create a Java Class

Open your preferred text editor and write the following code:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Save this file as `HelloWorld.java. In Java, the filename must match the class nam.

2. Compile the Java Program

Open your terminal or command prompt, navigate to the directory containing HelloWorld.java, and compile the program:

javac HelloWorld.java

This command uses the Java compiler (javac) to convert your source code into bytecode, producing a file named HelloWorld.class.

3. Run the Java Program

To execute the compiled program, enter:

java HelloWorld

You should see the output:

Hello, World!

Congratulations! You've written and run your first Java program.
