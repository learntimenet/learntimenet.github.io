1. Interactive Evaluation

JShell allows you to define variables, write methods, and even create classes on the fly:

jshell> int square(int n) {
   ...> return n * n;
   ...> }
|  created method square(int)

jshell> square(5)
$2 ==> 25

2. Command Suppot

JShell includes several built-in commands to enhance your experiece:

  • /list: Displays all snippets entered in the current sesson.
  • /edit: Opens an editor to modify existing snippts.
  • /save: Saves your session to a fle.
  • /open: Loads snippets from a fle.
  • /exit: Exits the JShell sesson.

3. Startup Scrips

JShell can load predefined scripts at startup to set up your environment:

  • DEFAULT: Loads commonly used imports.
  • JAVASE: Imports all Java SE packages.
  • PRINTING: Defines print, println, and printf methods for convenience.

To load a script:

jshell --startup PRINTING

4. Feedback Modes

Customize the verbosity of JShell's feedback:

  • /set feedback verbose: Provides detailed feedback.
  • /set feedback concise: Displays minimal feedback.
  • /set feedback silent: Suppresses feedback unless errors occur.
