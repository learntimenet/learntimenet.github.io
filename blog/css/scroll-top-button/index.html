Introduction

In this article, we're going to learn how to create a button to go to the top of the page using HTML, CSS and JavaScript. With easy-to-understand source code, you can follow the steps and create your own version.

Code source

code-tabs

  • html

    <button class="scroll-top-button">
  <i class="fa fa-chevron-up arrow"></i>
</button>

  • css

    .scroll-top-button {
  background-color: #ff9800;
  padding: 15px;
  border: none;
  border-radius: 5px;
  position: fixed;
  bottom: 20px;
  right: 20px;
  cursor: pointer;
  transition: background-color .3s, opacity .3s;
  /* Show Button only when scrolling to the buttom */
  opacity: 0;
}
.arrow {
  color: #fff;
}
.scroll-top-button:hover {
  background-color: #C67703;
}

  • javascript

    const scrollTopButton = document.querySelector(".scroll-top-button");

scrollTopButton.addEventListener('click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});

document.addEventListener('scroll', (event) => {
  if (window.pageYOffset < 100) {
    scrollTopButton.style.opacity = 0;
  } else {
    scrollTopButton.style.opacity = 1;
  }
});
