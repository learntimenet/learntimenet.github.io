üéØ What Is HAVING?

The HAVING clause is used to filter groups created by the GROUP BY clause.

While the WHERE clause filters individual rows, HAVING allows you to filter the result of aggregated data.

üß± HAVING Syntax

SELECT column, AGGREGATE_FUNCTION(column)
FROM table
GROUP BY column
HAVING AGGREGATE_FUNCTION(column) condition;

üß™ Example: Products with Total Sales Above a Threshold

SELECT product_id, SUM(amount) AS total_sales
FROM sales
GROUP BY product_id
HAVING SUM(amount) > 1000;

This returns only the products that have total sales greater than 1000.

üß™ Example: Customers with More Than 3 Orders

SELECT customer_id, COUNT(*) AS order_count
FROM orders
GROUP BY customer_id
HAVING COUNT(*) > 3;

This will return only customers who have placed more than three orders.
