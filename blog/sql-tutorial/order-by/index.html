üìä What Is ORDER BY?

The ORDER BY clause is used to sort the result set of a query by one or more columns. You can sort results in ascending (ASC) or descending (DESC) order.

  • DESC ‚Äì descending order

üîº Example: Ascending vs Descending

üìö Sorting by Multiple Columns

You can sort by more than one column. SQL will sort by the first column, and for rows with the same value, sort by the second, and so on.

SELECT name, department, salary
FROM employees
ORDER BY department ASC, salary DESC;

üí° This sorts alphabetically by department, and then within each department, by highest salary.

Different SQL databases handle NULL values differently when sorting:

You can often control this behavior:
