‚ú® What Is SELECT DISTINCT?

The SELECT DISTINCT statement is used to return only unique (different) values from a table. It eliminates duplicate rows in the result set.

This is especially useful when you're analyzing categories, tags, countries, or any dataset where duplicates might exist.

  • It applies to all selected columns
