üéØ What Is the WHERE Clause?

The WHERE clause is used to filter records based on specific conditions. It allows you to return only the rows that match your criteria.

You can use it in statements like:

  • SELECT ‚Äì filter the rows you retrieve
  • UPDATE ‚Äì choose which rows to change
  • DELETE ‚Äì specify which rows to remove

üß± Basic Syntax

SELECT column1, column2
FROM table_name
WHERE condition;

Example:

SELECT name, country
FROM customers
WHERE country = 'Canada';

üß† Logical Operators

You can combine multiple conditions using:

  • AND ‚Äì All conditions must be true
  • OR ‚Äì At least one condition must be true
  • NOT ‚Äì Negates a condition

Example:

FROM employees
WHERE age > 30 AND department = 'Sales';


SELECT name
FROM users
WHERE NOT country = 'USA';

üß™ Real-World Examples

Users aged between 20 and 30
