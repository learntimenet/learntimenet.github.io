📊 What Is GROUP BY?

GROUP BY is used in SQL to aggregate data by one or more columns.

It groups rows that have the same values into summary rows, like:

  • Total sales per product
  • Count of users per country
  • Average ratings per item

It’s typically used with aggregate functions like COUNT(), SUM(), AVG(), MIN(), or MAX().

🧱 GROUP BY Syntax

SELECT column, AGGREGATE_FUNCTION(column)
FROM table
GROUP BY column;

🧪 Example: Count Orders Per Customer

SELECT customer_id, COUNT(*) AS order_count
FROM orders
GROUP BY customer_id;

Result: A list of each customer with the total number of their orders.

🧪 Example: Total Sales Per Product

SELECT product_id, SUM(amount) AS total_sales
FROM sales
GROUP BY product_id;

Great for dashboards or reports.

🧪 Example: Average Rating Per Movie

SELECT movie_id, AVG(rating) AS average_rating
FROM reviews
GROUP BY movie_id;

🎯 When to Use GROUP BY

Use GROUP BY when:

  • You want summary data from a large set of rows
  • You're creating reports, dashboards, or analytics views
  • You’re using aggregate functions and want the output organized

📘 Recap

  • GROUP BY organizes rows into groups based on shared values
  • Use with functions like SUM(), AVG(), COUNT()
  • Essential for data analysis and reporting in SQL
