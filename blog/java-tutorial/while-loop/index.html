üîÑ What is a While Loop?

A while loop keeps running as long as its condition is true.

üîß Syntax:

while (condition) {
    // code to repeat
}

‚úÖ Example

int i = 1;

while (i <= 5) {
    System.out.println("i = " + i);
    i++;
}

üîÅ This prints i = 1 through i = 5.

üîÇ The do...while Loop

This version always runs at least once ‚Äî because the condition is checked after the first execution.

üîß Syntax:

do {
    // code to run
} while (condition);

Example:

int i = 1;

do {
    System.out.println("i = " + i);
    i++;
} while (i <= 5);

üîç When to Use Which?

Use this... When...
while You may want to skip the loop entirely
do...while You always want it to run at least once

‚ö†Ô∏è Common Mistakes

  • Forgetting to update the loop variable (infinite loop!)
  • Misplacing semicolons:
    while (i < 5); // ‚ùå Don't do this
  • Using assignment instead of comparison:
    while (x = 5) // ‚ùå should be while (x == 5)

üìò Recap
