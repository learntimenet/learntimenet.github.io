üß± What is an Array?

An array is a container that holds a fixed number of values of the same type.

Think of it as a row of boxes:

int[] numbers = {10, 20, 30, 40};

üîß Declaring Arrays

There are a couple of ways to declare arrays in Java:

int[] scores;      // preferred
String names[];    // also valid

Then initialize:

scores = new int[5];  // space for 5 integers

Or declare and initialize in one line:

int[] ages = {18, 21, 25, 30};

üì• Accessing Array Elements

Arrays are zero-indexed:

System.out.println(ages[0]);  // 18
ages[2] = 26;                 // update value

üîÅ Looping Through Arrays

Use a for loop to go through each element:

for (int i = 0; i < ages.length; i++) {
    System.out.println(ages[i]);
}

Or a foreach loop:

for (int age : ages) {
    System.out.println(age);
}

üìê Array Length

Use .length to get the number of elements:

System.out.println("Total: " + ages.length);

üî¢ Multidimensional Arrays

You can create 2D (or more) arrays:

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};

System.out.println(matrix[1][2]);  // Output: 6

Looping through 2D arrays:

for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}

üß† Example: Average Calculator

public class Average {
    public static void main(String[] args) {
        int[] marks = {85, 90, 78, 92};
        int sum = 0;

        for (int mark : marks) {
            sum += mark;
        }

        double average = sum / (double) marks.length;
        System.out.println("Average: " + average);
    }
}
