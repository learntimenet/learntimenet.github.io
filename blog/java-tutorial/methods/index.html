A method is a named block of code you can run (or "call") when needed. It's like a function in other languages.

üß™ Basic Syntax

    // code

public class Greet {
    // Method definition
    static void sayHello() {
        System.out.println("Hello!");
    }

    public static void main(String[] args) {
        // Method call
        sayHello();
    }
}

üì• Method Parameters

You can pass values (arguments) into methods.

static void greet(String name) {
    System.out.println("Hello, " + name + "!");
}

// Calling it:
greet("Java"); // Output: Hello, Java!

üéØ Return Values

Methods can return values using the return keyword.

static int add(int a, int b) {
    return a + b;
}

// Use the result
int sum = add(3, 4);  // sum = 7

Always match the return type in your method declaration.

üß¨ Method Overloading

You can define multiple methods with the same name, as long as their parameters differ.

static int multiply(int a, int b) {
    return a * b;
}

static double multiply(double a, double b) {
    return a * b;
}

The compiler chooses which method to use based on the arguments.

üí° Why Use Methods?

  • Reusability ‚Äì write once, use many times
  • Readability ‚Äì break large tasks into smaller steps
  • Maintainability ‚Äì fix bugs or update logic in one place
