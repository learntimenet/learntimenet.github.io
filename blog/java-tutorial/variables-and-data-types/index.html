üî§ What is a Variable?

A variable is a container for data. It holds a value you can use and change throughout your program.

Basic syntax:

type name = value;

Example:

int age = 25;

üî¢ Primitive Data Types

Java has 8 built-in primitive types:

Type Size Example
int 4 bytes int x = 10;
double 8 bytes double pi = 3.14;
char 2 bytes char letter = 'A';
boolean 1 bit boolean flag = true;
byte 1 byte byte b = 100;
short 2 bytes short s = 500;
long 8 bytes long l = 100000L;
float 4 bytes float f = 3.14f;

üß™ Reference Types

These refer to objects, not simple values.

Example:

String name = "Java";

Common reference types:

  • String
  • Arrays (e.g., int[] numbers = {1, 2, 3};)
  • Custom classes

üß† Declaring & Initializing Variables

You can declare first and assign later:

int score;
score = 100;

You can also declare multiple variables of the same type:

int x = 1, y = 2, z = 3;

üîÅ Type Casting

Implicit (widening):

int x = 10;
double y = x;  // OK, no data loss

Explicit (narrowing):

double pi = 3.14;
int rounded = (int) pi;  // You lose decimals

üìò Recap

  • Java has primitive types for simple values and reference types for objects.
  • Variables store data that you can reuse and modify.
  • Use final for values that should never change.
