üîç What Are Annotations?

Annotations are metadata added to Java code that give the compiler or frameworks extra information.

They don't affect code execution directly but help tools like the compiler, IDEs, and libraries.

‚úÖ Common Built-In Annotations

@Override

Tells the compiler you're overriding a method from a superclass.

@Override
public String toString() {
    return "Custom String";
}

‚úÖ Helps catch typos or signature mismatches.

@Deprecated

Marks a method/class as outdated. A warning is shown if it‚Äôs used.

@Deprecated
public void oldMethod() {
    System.out.println("Use newMethod() instead");
}

@SuppressWarnings

Instructs the compiler to ignore specific warnings.

@SuppressWarnings("unchecked")
ArrayList list = new ArrayList();  // avoids raw type warning

@FunctionalInterface

Ensures that an interface has only one abstract method (used in lambdas).

@FunctionalInterface
interface MyFunc {
    void execute();
}

‚úçÔ∏è Creating Custom Annotations

@interface MyAnnotation {
    String value();
}

Use it like this:

@MyAnnotation("Hello!")
public class Demo {}

You can also set default values:

@interface Info {
    String author() default "Unknown";
    int version() default 1;
}

üß† Where Are Annotations Used?

Annotations are everywhere in modern Java development:

  • JUnit ‚Äì @Test, @Before, @After
  • Spring Framework ‚Äì @Autowired, @Service, @Controller
  • Serialization libraries ‚Äì like Jackson: @JsonProperty
  • JPA (Hibernate) ‚Äì @Entity, @Id, @Column

They help configure behavior without writing extra code.

‚öôÔ∏è Retention & Target

Advanced annotations can define where and how long they apply:

@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
public @interface RunThis {}
  • Retention = how long the annotation is kept (e.g., in bytecode or at runtime)
  • Target = where it can be used (class, method, field, etc.)

  • Annotations are metadata that help tools and frameworks
  • Java has many built-in ones like @Override, @Deprecated
  • You can define custom annotations for flexibility
  • Widely used in modern frameworks like Spring, JPA, JUnit
