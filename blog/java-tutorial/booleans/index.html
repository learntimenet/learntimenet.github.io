üîç What is a Boolean?

A boolean is a data type that can hold only two values:

true
false

It‚Äôs used to represent yes/no, on/off, true/false logic in your code.

You can assign boolean values directly or as the result of expressions.

üî¢ Boolean Expressions

Boolean values often come from comparing values:

Other examples:

x != y     // true
x >= 5     // true

üß† Using Booleans in if Statements

    // do something
