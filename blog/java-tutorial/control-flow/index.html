🧭 Conditional Statements

These allow your program to make decisions.

✅ if, else if, else

int score = 85;

if (score >= 90) {
    System.out.println("Grade: A");
} else if (score >= 80) {
    System.out.println("Grade: B");
} else {
    System.out.println("Grade: C or lower");
}

🔄 switch Statement

Best for checking one variable against many possible values.

int day = 3;

switch (day) {
    case 1: System.out.println("Monday"); break;
    case 2: System.out.println("Tuesday"); break;
    case 3: System.out.println("Wednesday"); break;
    default: System.out.println("Other day");
}

⚠️ Use break to avoid fall-through (executing multiple cases).

🔁 Loops

Loops let you repeat code while a condition is true.

➰ for Loop (Count-Controlled)

for (int i = 1; i <= 5; i++) {
    System.out.println("Count: " + i);
}

🔄 while Loop (Condition-Controlled)

int i = 1;
while (i <= 5) {
    System.out.println("Count: " + i);
    i++;
}

🔂 do-while Loop (Runs at Least Once)

int i = 1;
do {
    System.out.println("Count: " + i);
    i++;
} while (i <= 5);

🧨 break and continue

break: exit a loop early

for (int i = 1; i <= 10; i++) {
    if (i == 5) break;
    System.out.println(i);
}

continue: skip current loop iteration

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);  // 3 is skipped
}

🧠 Example: Login Attempt Logic

String password = "java123";
int attempts = 0;
boolean loggedIn = false;

while (attempts < 3 && !loggedIn) {
    // simulate input
    String input = "java123";
    if (input.equals(password)) {
        System.out.println("Welcome!");
        loggedIn = true;
    } else {
        System.out.println("Wrong password.");
        attempts++;
    }
}

📘 Recap

  • Use if, else, and switch to handle decisions
  • Use loops to repeat code: for, while, do-while
  • Control loop behavior with break and continue
