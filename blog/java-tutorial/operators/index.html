üß† Operator Precedence

Just like math, Java follows an order of operations:

  1. ()
  2. *, /, %
  3. +, -
  4. Comparisons (>, <, etc.)
  5. Logical (&&, ||)
  6. Assignment (=)

Example:

int result = 2 + 3 * 4; // result = 14, not 20!

Use () to control order:

int result = (2 + 3) * 4; // result = 20
