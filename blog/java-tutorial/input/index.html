üß∞ Importing Scanner

To use input features, you need to import the Scanner class from java.util.

üî° Basic Example

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter your name: ");
        String name = sc.nextLine();
        
        System.out.println("Hello, " + name + "!");
    }
}

üìå Other Input Types

Method Input Type
next() Single word
nextLine() Whole line
nextInt() Integer
nextDouble() Decimal
nextBoolean() true / false

‚ö†Ô∏è Common Pitfall: Mixing nextInt() and nextLine()

int age = sc.nextInt();
sc.nextLine(); // Consume leftover newline
String name = sc.nextLine();

Always clear the buffer after nextInt() or nextDouble() before using nextLine().

üß™ Sample Program

üßº Closing Scanner

Don't forget to close the scanner when you're done:

üìò Recap

  • Use Scanner for console input
  • Choose the correct method for data type
  • Use nextLine() for full sentences
  • Watch out for input buffer issues!
