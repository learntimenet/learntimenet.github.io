üí° What is a Lambda Expression?

A lambda expression is a short block of code that takes in parameters and returns a result ‚Äî kind of like a method without a name.

Introduced in Java 8, it supports functional programming style in Java.

üìè Syntax

(parameters) -> expression

Or for multiple statements:

(parameters) -> {
    // code
    return result;
}

üß™ Simple Example

Runnable r = () -> System.out.println("Running a thread");
r.run();

‚úÖ Replaces boilerplate of anonymous inner classes
‚úÖ Improves readability and conciseness

üîÅ With Functional Interfaces

A lambda expression can be used only where there‚Äôs a functional interface ‚Äî one with just one abstract method.

@FunctionalInterface
interface Greet {
    void sayHello();
}

public class Demo {
    public static void main(String[] args) {
        Greet g = () -> System.out.println("Hello!");
        g.sayHello();
    }
}

üß† Common Functional Interfaces (from java.util.function)

Interface Method Description
Consumer<T> accept(T t) Takes input, returns nothing
Supplier<T> get() No input, returns something
Function<T,R> apply(T t) Takes input, returns output
Predicate<T> test(T t) Takes input, returns boolean

üßπ Lambdas with Collections

forEach()

List<String> names = List.of("Alice", "Bob", "Charlie");
names.forEach(name -> System.out.println(name));

filter(), map(), collect()

List<String> names = List.of("Anna", "Bella", "Chris");

List<String> result = names.stream()
    .filter(name -> name.startsWith("A"))
    .map(String::toUpperCase)
    .collect(Collectors.toList());

System.out.println(result); // [ANNA]

Streams + lambdas = magic ‚ú®

üîÑ Method References

Shorter way to write lambdas when you're calling existing methods:

names.forEach(System.out::println);

üìò Recap

  • Lambdas make Java code more expressive and concise
  • Work with functional interfaces (like Runnable, Predicate)
  • Combine with collections and streams for clean data processing
  • Use method references for extra brevity
