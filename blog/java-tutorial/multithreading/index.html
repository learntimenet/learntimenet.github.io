‚öôÔ∏è What is Multithreading?

Multithreading means running multiple threads of execution simultaneously ‚Äî like multitasking for your program.

‚úÖ Improves performance
‚úÖ Handles heavy tasks without freezing the UI
‚úÖ Essential for real-time, responsive applications

üëØ‚Äç‚ôÇÔ∏è Creating a Thread ‚Äì Two Ways

1. Extending the Thread class

class MyThread extends Thread {
    public void run() {
        System.out.println("Thread running");
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread t1 = new MyThread();
        t1.start();  // starts new thread
    }
}

2. Implementing Runnable

class MyRunnable implements Runnable {
    public void run() {
        System.out.println("Runnable running");
    }
}

public class Main {
    public static void main(String[] args) {
        Thread t2 = new Thread(new MyRunnable());
        t2.start();
    }
}

‚úÖ Runnable is preferred in modern Java for flexibility and cleaner code.

üîÑ Running Threads in Parallel

class Task implements Runnable {
    private String name;

    public Task(String name) {
        this.name = name;
    }

    public void run() {
        for (int i = 1; i <= 3; i++) {
            System.out.println(name + " - Step " + i);
        }
    }
}

public class Main {
    public static void main(String[] args) {
        Thread t1 = new Thread(new Task("Task A"));
        Thread t2 = new Thread(new Task("Task B"));

        t1.start();
        t2.start();
    }
}

üõ°Ô∏è Synchronization ‚Äì Avoiding Conflicts

Use synchronized:

public synchronized void increment() {
    count++;
}
