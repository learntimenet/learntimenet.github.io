Introduction

Angular is a popular front-end framework for building dynamic web applications. This cheat sheet provides a quick reference for Angular commands, syntax, and best practices.

Angular CLI Commands

The Angular CLI is a powerful tool for scaffolding and managing Angular projects.

cheatsheet

  • Creates a new Angular project
    ng new project-name
  • Starts a development server (http://localhost:4200)
    ng serve
  • Runs the app on a custom port
    ng serve --port 5000
  • Compiles the app into dist/ folder
    ng build
  • Production build (optimized)
    ng build --prod
  • Runs unit tests (Karma + Jasmine)
    ng test
  • Runs end-to-end tests (Protractor)
    ng e2e
  • Generates a new component
    ng generate component my-component
  • Generates a new service
    ng generate service my-service
  • Generates a new module
    ng generate module my-module
  • Generates a new directive
    ng generate directive my-directive
  • Generates a new pipe
    ng generate pipe my-pipe
  • Adds Angular Material to the project
    ng add @angular/material
  • Updates Angular packages
    ng update
  • Runs linting (TSLint/ESLint)
    ng lint

Component Basics

Template Syntax

<!-- Interpolation -->
<p>{{ title }}</p>

<!-- Property Binding -->
<img [src]="imageUrl">

<!-- Event Binding -->
<button (click)="onClick()">Click Me</button>

<!-- Two-Way Binding (requires FormsModule) -->
<input [(ngModel)]="name">

Component Structure

import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.html',
  styleUrls: ['./my-component.css']
})
export class MyComponent {
  title = 'My App';
  
  onClick() {
    console.log('Button clicked!');
  }
}

Directives

Built-in Directives

cheatsheet

  • Conditionally renders an element
    *ngIf
  • Loops over a list
    *ngFor
  • Dynamically applies CSS classes
    [ngClass]
  • Dynamically applies inline styles
    [ngStyle]
  • Switch-case for templates
    [ngSwitch]

Example:

<div *ngIf="isLoggedIn">Welcome back!</div>

<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>

<div [ngClass]="{ 'active': isActive }"></div>

<div [ngStyle]="{ 'color': textColor }"></div>

<div [ngSwitch]="role">
  <p *ngSwitchCase="'admin'">Admin Panel</p>
  <p *ngSwitchCase="'user'">User Dashboard</p>
  <p *ngSwitchDefault>Guest</p>
</div>

Services & Dependency Injection

Creating a Service

import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root' // Singleton service
})
export class DataService {
  getData() {
    return ['Item 1', 'Item 2', 'Item 3'];
  }
}

Using a Service in a Component

import { Component } from '@angular/core';
import { DataService } from './data.service';

@Component({...})
export class MyComponent {
  items: string[];

  constructor(private dataService: DataService) {
    this.items = this.dataService.getData();
  }
}

HTTP Client (API Calls)

Import HttpClientModule in app.module.ts:

import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule],
})
export class AppModule {}

Use HttpClient in a Service:

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable()
export class ApiService {
  constructor(private http: HttpClient) {}

  getUsers() {
    return this.http.get('https://api.example.com/users');
  }

  postData(data: any) {
    return this.http.post('https://api.example.com/data', data);
  }
}

Subscribe in a Component:

this.apiService.getUsers().subscribe(users => {
  console.log(users);
});

Routing & Navigation

Define Routes in app-routing.module.ts:

import { RouterModule, Routes } from '@angular/router';
import { HomeComponent } from './home.component';
import { AboutComponent } from './about.component';

const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: '**', redirectTo: '' } // Wildcard route (404)
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule {}

Add <router-outlet> in app.component.html:

<router-outlet></router-outlet>

Navigate Programmatically:

import { Router } from '@angular/router';

constructor(private router: Router) {}

goToAbout() {
  this.router.navigate(['/about']);
}

Link in Template:

<a routerLink="/">Home</a>
<a routerLink="/about">About</a>

Forms

Template-Driven Forms

Import FormsModule in your app.module.ts:

import { FormsModule } from '@angular/forms';

@NgModule({
  imports: [FormsModule],
})
export class AppModule {}

Use ngModel and ngSubmit in your components: 

<form #myForm="ngForm" (ngSubmit)="onSubmit(myForm)">
  <input name="username" ngModel required>
  <button type="submit">Submit</button>
</form>

Reactive Forms

Import ReactiveFormsModule in your app.module.ts:

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [ReactiveFormsModule],
})
export class AppModule {}

Create your FormGroup in your component:

export class MyFormComponent {
  form = new FormGroup({
    username: new FormControl('', Validators.required),
    email: new FormControl('', [Validators.required, Validators.email])
  });

  onSubmit() {
    console.log(this.form.value);
  }
}

And use it in the html code of your component:

<form [formGroup]="form" (ngSubmit)="onSubmit()">
  <input formControlName="username">
  <input formControlName="email">
  <button type="submit">Submit</button>
</form>
