üîç What Are Data Types?

In JavaScript, a data type is a classification of the type of value a variable can hold. It helps JavaScript understand what kind of operations can be performed on a particular value. JavaScript has several built-in data types.

üß© Types of Data Types in JavaScript

JavaScript has two categories of data types:

  1. Primitive Data Types (also known as basic types)
  2. Non-Primitive Data Types (also called reference types)

Let‚Äôs dive into both.

1Ô∏è‚É£ Primitive Data Types

Primitive data types are the simplest forms of data in JavaScript. They are immutable (cannot be changed after creation).

1.1 String

A string is a sequence of characters used to represent text. Strings can be enclosed in single quotes ('), double quotes ("), or backticks (`).

let greeting = "Hello, World!";
let name = 'Alice';

1.2 Number

A number represents both integer and floating-point values.

let age = 25;  // integer
let price = 19.99;  // floating-point number

1.3 Boolean

A boolean has two possible values: true or false.

let isActive = true;
let isCompleted = false;

1.4 Undefined

An undefined value means a variable has been declared but not assigned a value.

let userName;
console.log(userName);  // Output: undefined

1.5 Null

Null is a special value representing the absence of any object value.

let person = null;

1.6 Symbol (ES6)

A symbol is a unique and immutable primitive value. Symbols are primarily used for object property keys.

let sym = Symbol('description');

1.7 BigInt (New in ES11)

A BigInt allows you to work with large integers (beyond the range of the Number type).

let bigNumber = 123456789123456789123456789n;  // BigInt

2Ô∏è‚É£ Non-Primitive Data Types

Non-primitive data types are objects that can hold collections of data and more complex entities.

2.1 Object

An object is a collection of key-value pairs. It‚Äôs used to store more complex data structures.

let person = {
  name: 'John',
  age: 30,
  isActive: true
};

2.2 Array

An array is an ordered collection of values (can be of any type). Arrays are a type of object.

let colors = ['red', 'green', 'blue'];

2.3 Function

A function is a block of code designed to perform a particular task. Functions are first-class objects in JavaScript.

function greet(name) {
  return "Hello, " + name;
}

let message = greet('Alice');  // "Hello, Alice"

üîÑ Type Conversion

JavaScript allows you to convert data from one type to another. This process is called type casting.

Implicit Type Conversion

JavaScript automatically converts data types in some operations.

let num = 10;
let str = "The number is " + num;  // Implicit conversion: 10 is converted to a string
console.log(str);  // "The number is 10"

Explicit Type Conversion

You can explicitly convert a value to a specific type using functions like String(), Number(), and Boolean().

let str = "123";
let num = Number(str);  // Explicit conversion: "123" is converted to 123
console.log(num);  // 123

üß† Recap

  • Primitive data types include string, number, boolean, undefined, null, symbol, and BigInt.
  • Non-primitive data types include objects, arrays, and functions.
  • JavaScript allows both implicit and explicit type conversion.
