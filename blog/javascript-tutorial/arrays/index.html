🔍 What Is an Array?

An array in JavaScript is a data structure used to store multiple values in a single variable. Unlike variables that hold a single value, arrays allow you to store ordered collections of data (of any type), such as numbers, strings, or objects.

Arrays in JavaScript are zero-indexed, meaning the first element is at position 0, the second at position 1, and so on.

Example:

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]);  // Output: Apple

In this example, fruits is an array containing three string elements.

🧑‍🍳 Creating an Array

You can create an array using square brackets ([]). The values inside the brackets are the elements of the array.

Example:

let numbers = [10, 20, 30, 40];
let mixedArray = [1, "Hello", true, null];

You can also create an empty array and add elements later:

let emptyArray = [];
emptyArray.push(1);
emptyArray.push(2);
console.log(emptyArray);  // Output: [1, 2]

🔄 Accessing Array Elements

You can access elements in an array using the index inside square brackets ([]).

Example:

let colors = ["Red", "Green", "Blue"];
console.log(colors[0]);  // Output: Red
console.log(colors[1]);  // Output: Green

If you try to access an index that does not exist, you will get undefined:

console.log(colors[5]);  // Output: undefined

🛠️ Array Methods

JavaScript provides many built-in methods to manipulate arrays. Here are some of the most commonly used ones:

1️⃣ .push()

The .push() method adds one or more elements to the end of an array.

let fruits = ["Apple", "Banana"];
fruits.push("Cherry");
console.log(fruits);  // Output: ["Apple", "Banana", "Cherry"]

2️⃣ .pop()

The .pop() method removes the last element from an array.

let fruits = ["Apple", "Banana", "Cherry"];
fruits.pop();
console.log(fruits);  // Output: ["Apple", "Banana"]

3️⃣ .shift()

The .shift() method removes the first element from an array.

let fruits = ["Apple", "Banana", "Cherry"];
fruits.shift();
console.log(fruits);  // Output: ["Banana", "Cherry"]

4️⃣ .unshift()

The .unshift() method adds one or more elements to the beginning of an array.

let fruits = ["Banana", "Cherry"];
fruits.unshift("Apple");
console.log(fruits);  // Output: ["Apple", "Banana", "Cherry"]

5️⃣ .splice()

The .splice() method allows you to remove or add elements at any position in the array.

Example (Removing Elements):

let fruits = ["Apple", "Banana", "Cherry"];
fruits.splice(1, 1);  // Removes 1 element starting from index 1
console.log(fruits);  // Output: ["Apple", "Cherry"]

Example (Adding Elements):

let fruits = ["Apple", "Banana"];
fruits.splice(1, 0, "Cherry");  // Adds "Cherry" at index 1 without removing any elements
console.log(fruits);  // Output: ["Apple", "Cherry", "Banana"]

6️⃣ .forEach()

The .forEach() method iterates over all elements in an array, executing a provided function on each element.

let fruits = ["Apple", "Banana", "Cherry"];
fruits.forEach(function(fruit) {
  console.log(fruit);
});
// Output:
// Apple
// Banana
// Cherry

7️⃣ .map()

The .map() method creates a new array with the results of calling a provided function on every element in the array.

let numbers = [1, 2, 3, 4];
let doubled = numbers.map(function(num) {
  return num * 2;
});
console.log(doubled);  // Output: [2, 4, 6, 8]

🧑‍🍳 Looping Over Arrays

There are various ways to loop through arrays in JavaScript, such as for, forEach(), and map().

Example with for Loop:

let fruits = ["Apple", "Banana", "Cherry"];
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
// Output:
// Apple
// Banana
// Cherry

Example with forEach():

let fruits = ["Apple", "Banana", "Cherry"];
fruits.forEach(function(fruit) {
  console.log(fruit);
});
// Output:
// Apple
// Banana
// Cherry

🔄 Nested Arrays

Arrays can contain other arrays as elements. These are called nested arrays.

Example:

let nestedArray = [[1, 2], [3, 4], [5, 6]];
console.log(nestedArray[0]);  // Output: [1, 2]
console.log(nestedArray[0][1]);  // Output: 2

You can access elements inside the nested array by using multiple indices.

💡 Array Destructuring

Array destructuring allows you to extract values from an array and assign them to variables.

Example:

let fruits = ["Apple", "Banana", "Cherry"];
let [first, second] = fruits;
console.log(first);  // Output: Apple
console.log(second); // Output: Banana

You can also skip elements and assign values to variables:

let fruits = ["Apple", "Banana", "Cherry"];
let [, , third] = fruits;
console.log(third); // Output: Cherry

🧠 Recap

  • Arrays in JavaScript are used to store multiple values in a single variable.
  • You can access array elements using their index, starting from 0.
  • JavaScript provides many array methods such as .push(), .pop(), .shift(), .unshift(), .splice(), .forEach(), and .map().
  • You can loop through arrays using for loops, forEach(), and map().
  • Nested arrays are arrays inside arrays, and you can use destructuring to extract values from them easily.
