üåê What Is JavaScript?

JavaScript is a lightweight, interpreted programming language most commonly used to build interactive websites. It works alongside HTML and CSS to create rich user experiences.

It's supported by all modern web browsers, and can be used for both frontend and backend development.

‚öôÔ∏è How JavaScript Works

JavaScript code is executed by the browser‚Äôs JavaScript engine. It can:

  • Respond to user actions (like clicks and keyboard input)
  • Modify HTML and CSS dynamically
  • Send and receive data using APIs

üöÄ Why Learn JavaScript?

  • It's the most widely-used programming language for the web
  • Enables real-time interaction in web applications
  • Works with popular frameworks like React, Angular, and Vue
  • Can also be used server-side with Node.js

üõ† Writing Your First Script

You can write JavaScript directly inside an HTML file using the <script> tag:

<!DOCTYPE html>
<html>
  <head>
    <title>My First JS</title>
  </head>
  <body>
    <h1>Hello JavaScript!</h1>
    
    <script>
      alert("Welcome to JavaScript!");
    </script>
  </body>
</html>

When you open this HTML file in a browser, a pop-up message will appear.

‚úçÔ∏è Basic Syntax Overview
