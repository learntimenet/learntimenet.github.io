üîç What Is a String?

A string is a sequence of characters used to represent text. Strings are one of the most commonly used data types in JavaScript, and they are enclosed in either single quotes ('), double quotes ("), or backticks (`).

Basic Syntax

let str1 = 'Hello, World!';
let str2 = "JavaScript is fun!";
let str3 = `This is a string with template literals!`;

üßë‚Äçüç≥ Creating Strings

You can create a string by enclosing characters in quotes:

Example:

let greeting = "Hello, World!";
let name = 'Alice';

You can use single quotes ('), double quotes ("), or backticks (`) to define strings. The choice depends on your needs, such as using quotes inside a string.

Example with Quotes Inside a String:

let sentence = "Alice said, 'Hello!'";
let anotherSentence = 'Bob replied, "Hi!"';

üß© String Concatenation

You can concatenate (combine) strings using the + operator.

Example:

let firstName = "John";
let lastName = "Doe";
let fullName = firstName + " " + lastName;
console.log(fullName);  // Output: John Doe

In this case, we joined firstName, a space, and lastName to create fullName.

You can also use template literals (backticks) to concatenate strings more easily:

let fullName = `${firstName} ${lastName}`;
console.log(fullName);  // Output: John Doe

üõ†Ô∏è String Methods

JavaScript strings come with a variety of built-in methods to perform common operations, like finding the length of a string, replacing characters, and converting case.

1Ô∏è‚É£ .length Property

The .length property returns the number of characters in a string.

let text = "Hello, World!";
console.log(text.length);  // Output: 13

2Ô∏è‚É£ .toUpperCase() and .toLowerCase()

These methods convert a string to uppercase or lowercase.

let greeting = "Hello!";
console.log(greeting.toUpperCase());  // Output: HELLO!
console.log(greeting.toLowerCase());  // Output: hello!

3Ô∏è‚É£ .indexOf()

The .indexOf() method returns the position of the first occurrence of a specified value in a string.

let text = "JavaScript is awesome!";
console.log(text.indexOf("is"));  // Output: 10

4Ô∏è‚É£ .slice()

The .slice() method extracts a portion of a string based on start and end positions.

let text = "JavaScript";
console.log(text.slice(0, 4));  // Output: Java

5Ô∏è‚É£ .replace()

The .replace() method replaces part of a string with a new value.

let sentence = "I like JavaScript";
let newSentence = sentence.replace("JavaScript", "programming");
console.log(newSentence);  // Output: I like programming

6Ô∏è‚É£ .split()

The .split() method splits a string into an array of substrings based on a separator.

let sentence = "Hello, World!";
let words = sentence.split(" ");
console.log(words);  // Output: ["Hello,", "World!"]

üìè String Escaping

To include special characters like quotes inside a string, you can use the escape character (\).

Example:

let quote = 'She said, "Hello!"';
let escapedQuote = "She said, \"Hello!\"";

Both will result in: She said, "Hello!"

üí° Template Literals

Template literals (backticks) allow you to create multi-line strings and embed expressions inside a string using ${}.

Example:

let name = "Alice";
let age = 25;

let greeting = `Hello, my name is ${name} and I am ${age} years old.`;
console.log(greeting);  // Output: Hello, my name is Alice and I am 25 years old.

üß† Recap

  • Strings in JavaScript represent text and are enclosed in quotes (', ", or backticks).
  • You can concatenate strings using + or template literals.
  • JavaScript provides many string methods, such as .length, .toUpperCase(), .indexOf(), and .replace().
  • Template literals are a convenient way to handle multi-line strings and embedded expressions.
