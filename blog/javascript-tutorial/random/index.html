üîç The Math.random() Method

JavaScript provides a built-in method called Math.random() to generate random numbers.

‚úÖ What It Does

  • Returns a floating-point number
  • Value is between 0 (inclusive) and 1 (exclusive)

üìå Example:

console.log(Math.random());  
// Output: e.g., 0.568901237

This result will always be 0 or greater and less than 1.

üéØ Random Integer Between 0 and N

To generate a random integer from 0 up to N - 1, use:

let random = Math.floor(Math.random() * N);

Example: Random number from 0 to 9

let random = Math.floor(Math.random() * 10);
console.log(random);  // Output: 0 to 9

üéØ Random Integer Between Two Numbers

To generate a number between two values (inclusive):

function getRandom(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

Example: Random number between 5 and 15

let result = getRandom(5, 15);
console.log(result);  // Output: a number from 5 to 15

üéÆ Use Cases for Math.random()

üî¢ 1. Generate Random IDs

üé∞ 2. Simulate Dice Rolls

üß† Recap

  • Use Math.random() to generate pseudo-random numbers between 0 and 1.
  • Multiply and Math.floor() the result to get whole numbers in a custom range.
  • Great for games, visual effects, or dynamic behavior.
