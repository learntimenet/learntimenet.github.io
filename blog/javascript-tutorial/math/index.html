üîÑ Random Numbers

üé≤ Math.random()

Returns a random decimal between 0 (inclusive) and 1 (exclusive):

To get a random number between 1 and 10:

console.log(random);  // Output: A number from 1 to 10

üî¢ Powers and Roots
