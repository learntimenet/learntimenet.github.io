üîç What Is a Number in JavaScript?

A number in JavaScript is a data type used to represent both integers (whole numbers) and floating-point numbers (decimals). JavaScript numbers are dynamic, meaning they are not explicitly defined with a data type in the code; JavaScript automatically treats numeric values as numbers.

Example:

let num1 = 42;      // An integer
let num2 = 3.14;    // A floating-point number (decimal)

üßë‚Äçüç≥ Working with Numbers

JavaScript allows you to perform a wide variety of mathematical operations on numbers. You can add, subtract, multiply, divide, and more.

Basic Arithmetic Operations:

let a = 10;
let b = 5;

console.log(a + b);  // Output: 15 (addition)
console.log(a - b);  // Output: 5  (subtraction)
console.log(a * b);  // Output: 50 (multiplication)
console.log(a / b);  // Output: 2  (division)

JavaScript has a few special numeric values that you may encounter when working with numbers:

1Ô∏è‚É£ Infinity

Infinity is a special value representing numbers that exceed the largest representable number in JavaScript.

NaN represents a value that is not a valid number. It typically occurs when a mathematical operation fails to produce a valid result.

console.log(invalidNumber);  // Output: NaN

üßÆ Mathematical Methods

JavaScript provides a built-in Math object with various methods for performing mathematical tasks like rounding numbers, finding square roots, and generating random numbers.

1Ô∏è‚É£ Math.round()

Rounds a number to the nearest integer.

let num = 3.6;
console.log(Math.round(num));  // Output: 4

2Ô∏è‚É£ Math.sqrt()

Returns the square root of a number.

let num = 16;
console.log(Math.sqrt(num));  // Output: 4

3Ô∏è‚É£ Math.random()

Generates a random number between 0 (inclusive) and 1 (exclusive).

let randomNum = Math.random();
console.log(randomNum);  // Output: A random number between 0 and 1

4Ô∏è‚É£ Math.max() and Math.min()

Returns the highest or lowest value among a set of numbers.

let max = Math.max(1, 5, 10, 2);
let min = Math.min(1, 5, 10, 2);

console.log(max);  // Output: 10
console.log(min);  // Output: 1

üîÑ Number Conversion

You might sometimes need to convert other data types to numbers. JavaScript provides a few ways to do this.

1Ô∏è‚É£ parseInt()

Converts a string to an integer.

let num = parseInt(str);
console.log(num);  // Output: 123

2Ô∏è‚É£ parseFloat()

Converts a string to a floating-point number.

let num = parseFloat(str);
console.log(num);  // Output: 3.14

3Ô∏è‚É£ Number()

Converts a value to a number.

let num = Number(value);
console.log(num);  // Output: 42

üí° Working with Large Numbers

JavaScript can handle large numbers, but precision may become an issue when dealing with extremely large or small values. To manage very large integers, JavaScript provides BigInt (a built-in object).

Example:

let bigNumber = 1234567890123456789012345678901234567890n;  // BigInt
console.log(bigNumber);  // Output: 1234567890123456789012345678901234567890n

Note the n at the end of the number, which indicates it is a BigInt.
