üîç What Are JavaScript Events?

JavaScript events are actions or occurrences that happen in the system you are programming for. They can be triggered by the user (like clicking a button or pressing a key) or by the browser (like the page loading or an image finishing loading). JavaScript can respond to these events with specific code.

Example of an Event:

<button onclick="alert('Button clicked!')">Click Me</button>

In this example, when the button is clicked, it triggers the onclick event, which displays an alert.

üõ†Ô∏è Event Listeners

An event listener is a more flexible way to handle events. You can attach an event listener to an element, which allows you to respond to events without inline code in the HTML.

Syntax:

element.addEventListener(event, function, useCapture);
  • event ‚Äì The type of event (e.g., click, mouseover)
  • function ‚Äì The function to run when the event occurs
  • useCapture ‚Äì Optional parameter to control event propagation (usually set to false)

Example:

<button id="myButton">Click Me</button>

<script>
  let button = document.getElementById("myButton");
  
  button.addEventListener("click", function() {
    alert("Button clicked using addEventListener!");
  });
</script>

In this example, when the button is clicked, the event listener triggers the alert.

üí° Event Propagation

JavaScript events propagate through the DOM in two phases: capturing and bubbling.

  • Capturing: The event starts from the root of the document and travels down to the target element.
  • Bubbling: The event starts at the target element and bubbles up to the root.

You can control this behavior with the useCapture parameter in addEventListener or by using stopPropagation() and preventDefault().

üß† Recap

  • Event propagation can be controlled through capturing and bubbling phases.
