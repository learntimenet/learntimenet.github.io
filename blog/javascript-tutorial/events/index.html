🔍 What Are JavaScript Events?

JavaScript events are actions or occurrences that happen in the system you are programming for. They can be triggered by the user (like clicking a button or pressing a key) or by the browser (like the page loading or an image finishing loading). JavaScript can respond to these events with specific code.

Example of an Event:

<button onclick="alert('Button clicked!')">Click Me</button>

In this example, when the button is clicked, it triggers the onclick event, which displays an alert.

📅 Event Types

There are many different types of events in JavaScript, and they can be categorized into several broad groups:

1️⃣ Mouse Events

  • click – Triggered when a mouse button is clicked
  • dblclick – Triggered when a mouse button is double-clicked
  • mouseover – Triggered when the mouse pointer enters an element
  • mouseout – Triggered when the mouse pointer leaves an element
  • mousemove – Triggered when the mouse moves over an element

Example:

<button onmouseover="console.log('Mouse entered button')">Hover over me</button>

2️⃣ Keyboard Events

  • keydown – Triggered when a key is pressed down
  • keyup – Triggered when a key is released
  • keypress – Triggered when a key is pressed (deprecated, use keydown or keyup)

Example:

<input type="text" onkeydown="console.log('Key pressed')">

3️⃣ Window/Document Events

  • load – Triggered when the page has finished loading
  • resize – Triggered when the window is resized
  • scroll – Triggered when the user scrolls the page
  • focus – Triggered when an element gains focus
  • blur – Triggered when an element loses focus

Example:

<script>
  window.onload = function() {
    console.log('Page loaded!');
  };
</script>

🛠️ Event Listeners

An event listener is a more flexible way to handle events. You can attach an event listener to an element, which allows you to respond to events without inline code in the HTML.

Syntax:

element.addEventListener(event, function, useCapture);
  • event – The type of event (e.g., click, mouseover)
  • function – The function to run when the event occurs
  • useCapture – Optional parameter to control event propagation (usually set to false)

Example:

<button id="myButton">Click Me</button>

<script>
  let button = document.getElementById("myButton");
  
  button.addEventListener("click", function() {
    alert("Button clicked using addEventListener!");
  });
</script>

In this example, when the button is clicked, the event listener triggers the alert.

🧑‍💻 Removing Event Listeners

You can also remove event listeners with the removeEventListener method.

Example:

<button id="myButton">Click Me</button>

<script>
  let button = document.getElementById("myButton");

  function handleClick() {
    alert("Button clicked!");
  }

  button.addEventListener("click", handleClick);

  // To remove the event listener:
  // button.removeEventListener("click", handleClick);
</script>

💡 Event Propagation

JavaScript events propagate through the DOM in two phases: capturing and bubbling.

  • Capturing: The event starts from the root of the document and travels down to the target element.
  • Bubbling: The event starts at the target element and bubbles up to the root.

You can control this behavior with the useCapture parameter in addEventListener or by using stopPropagation() and preventDefault().

🧠 Recap

  • JavaScript events are actions that occur due to user interaction or system events.
  • You can handle events using inline event attributes (e.g., onclick) or using event listeners.
  • Common event types include mouse events, keyboard events, and window/document events.
  • Event propagation can be controlled through capturing and bubbling phases.
