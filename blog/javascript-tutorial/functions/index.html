üîç What Are Functions?

A function is a block of code that is designed to perform a specific task. Functions allow you to group code that you can reuse throughout your program. You can think of a function as a "recipe" that performs an action when called.

Basic Syntax

function functionName() {
  // Code to be executed
}

üßë‚Äçüç≥ Defining a Function

You define a function using the function keyword followed by the function name, parentheses for parameters (optional), and a block of code inside curly braces {}.

Example:

function greet() {
  console.log("Hello, world!");
}

To execute (or call) the function, simply use the function name followed by parentheses:

greet();  // Output: Hello, world!

üìú Function Parameters

A function can accept parameters (input values) to make it more flexible.

Example:

function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice");  // Output: Hello, Alice!
greet("Bob");    // Output: Hello, Bob!

Here, name is a parameter, and you provide a value when calling the function.

üîÑ Function Return Values

A function can return a value, allowing you to use the result of the function in other parts of your code.

Example:

function add(a, b) {
  return a + b;
}

let sum = add(3, 4);  // sum will be 7
console.log(sum);      // Output: 7

The return keyword sends back the value to where the function was called.

Arrow functions are a shorter syntax for writing functions introduced in ES6.

4Ô∏è‚É£ Immediately Invoked Function Expression (IIFE)

An IIFE is a function that is called immediately after it is defined.

üéØ Function Scope

A function has its own scope, meaning that variables declared inside the function are only accessible within the function.

function myFunction() {
  let localVar = "I am local!";
  console.log(localVar);  // Output: I am local!
}

console.log(localVar);  // ‚ùå ReferenceError: localVar is not defined

Here, localVar is only accessible inside myFunction().

üß† Recap

  • A function is a block of code that performs a specific task.
  • Parameters allow you to pass values to a function when calling it.
  • Functions can return values using the return keyword.
  • There are different ways to define functions: declarations, expressions, arrow functions, and IIFEs.
  • Functions have their own scope.
