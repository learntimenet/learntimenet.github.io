Introduction

In this article, we'll learn about SQL join types, including inner join, left join, right join and full outer join. We'll cover their syntax, differences and practical examples for a better understanding.

INNER JOIN

Combines rows from two tables based on a common column between them

A
B 
SELECT *
FROM A
INNER JOIN B ON A.key = B.key

LEFT JOIN

Returns all rows in the left table and the corresponding rows in the right table according to a specified condition. If there is no match in the right-hand table, NULL values are returned for the columns in the right-hand table

A
B 
SELECT *
FROM A
LEFT JOIN B ON A.key = B.key

LEFT EXCLUDING JOIN

Returns all rows in the left-hand table that do not match any rows in the right-hand table

A
B 
SELECT *
FROM A
LEFT JOIN B ON A.key = B.key
WHERE B.key IS NULL

RIGHT JOIN

Returns all the rows in the right-hand table and the corresponding rows in the left-hand table according to a specified condition. If there is no match in the left-hand table, NULL values are returned for the columns in the left-hand table

A
B 
SELECT *
FROM A
RIGHT JOIN B ON A.key = B.key

RIGHT EXCLUDING JOIN

Returns all rows in the right-hand table that do not match any rows in the left-hand table.

B 
SELECT *
FROM A
RIGHT JOIN B ON A.key = B.key
WHERE A.key IS NULL

OUTER JOIN

Combines rows from two or more tables according to a specified condition, and includes non-matching rows from one or both tables
